1-Extrair os ficheiros
2-Verificar se estão os ficheiros "Position.java", "Node.java", "Comp.java" e "Jogo.java"
3-Fazer o comando abaixo para compilar o programa
	javac Jogo.java
4-Correr o programa com o comando
	java Jogo
5-Seguir os paços indicados no program
6-Jogar utilizando a configuração no exemplo abaixo
	    0   1   2
	   -----------
	0 | 1 | 2 | 3 |
	  -----------
	1 | 4 | 5 | 6 |
	  -----------
	2 | 7 | 8 | 9 |
 	   -----------
Este programa foi compilado em java com o Eclipse utilizando o linux 14.04
